Help:

1. Choose your source

2. Choose or create a Configuration

3 Click "Import Data"

4. Pay Attention to the Log

5. Wait until the data is Uploaded

6. Rows with errors can be downloaded with the "Get Skipped Rows" button

At the setup of this software you were asked for a path. That is the path where you can find the "server-url.cfg" file. Open this file with an editor of your choice (eg Notepad++) and edit the server url. Afterwards restart the server.

Depending on which file you choose there will be different steps to work off.

1. Your source is a .csv file

1.1 Delimiter:
The delimiter separates the columns of a csv file. Search the delimiter and write it into the textarea. An often used delimiter is ";".

1.2 Number of header lines:
Mostly there are some lines at the top of your table that you don't want to import. Count this lines and write this number in the textarea.

1.3 Current mapping:
If there are values in a column you want to replace, than you can use the mapping. Choose a column where you want to apply the mapping. At "old value" you write the value you want to replace. At "Replaced by" the value you want to replace the old value.

1.4 Datetime: Choose a timezone and columns where the date and time can be found. Enter a format, so that the program can recognize the date and time

1.5 Select or create a thing

1.6 Select or create a datastream

2. Your source is an excel file

1.1 Delimiter:
A delimiter is not needed.

1.2 Number of header lines:
Mostly there are some lines at the top of your table that you don't want to import. Count this lines and write this number in the textarea.

1.3 Current mapping:
If there are values in a column you want to replace, than you can use the mapping. Choose a column where you want to apply the mapping. At "old value" you write the value you want to replace. At "Replaced by" the value you want to replace the old value.

1.4 Datetime: Choose a timezone and a column where the date and time can be found. You don't need to enter a format.

1.5 Select or create a thing

1.6 Select or create a datastream

The preview shows the the first 3 lines of your file. It may help you to create a Configuration.

A file in "Get Skipped Rows" includes all lines, which can't be imported in the FROST-Server. You may want to edit your Configuration and try to upload this missing lines.

In the mapping table you can save if you want to replace a value with an other value in a column. The column scope you can enter a column where you want to apply the mapping. The old value is the value you want to replace. In the "replaced by" scope you enter the value which will replace the old value.

An example: You want to replace every value "midnight" with the value "00:00:00" in column 5.
What you have to do:
1. Add a row/mapping in the mapping popup
2. Column = 5
3. Old value = "midnight"
4. Replaced by = "00:00:00"
5. Click save
6. Close popup
